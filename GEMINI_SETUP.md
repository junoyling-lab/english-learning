# Gemini Setup

## Get an API Key

1. Go to Google AI Studio: https://aistudio.google.com/
2. Sign in with your Google account.
3. Create or copy a Gemini API key.
4. Open this app.
5. Paste the key into the API key field.
6. Save it.

## Notes

- The key is stored in this browser on this device.
- Do not publish this app with your API key inside the code.
- Free Gemini API limits can change.
- If you later share the app with other people, move the key behind a small backend such as Cloudflare Worker.
