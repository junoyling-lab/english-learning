# Gemini English Learning Assistant

A personal mobile-first PWA for learning English with Gemini while keeping your own vocabulary database.

The app starts with the user's requested default learning rules. You can change, show, or clear rules through the chat box after opening the app. If you clear rules, the app respects that and does not automatically restore defaults.

## What It Does

- Chat with Gemini about English usage, examples, workplace expressions, and word differences.
- Ask Gemini to create new words only when you request them.
- Save structured vocabulary summaries into a local wordbook.
- Delete, update, mark important, and review words from chat.
- Search your own wordbook by meaning, scenario, notes, tags, and examples.
- Export and import your local data backup.

## Daily Use

1. Open `index.html` in a browser, or host the folder and install it to your phone home screen.
2. Tap `API Key`, add your Gemini API key once, and save it.
3. Start chatting.
4. Optional examples:
   - `显示当前规则`
   - `以后每个词给 5 个例句`
   - `给我 5 个职场单词`
   - `总结并存到单词本`
   - `把 negotiate 标记为重要`
   - `删除 negotiate`
   - `帮我复习重要词`

## Privacy

Vocabulary data and your Gemini API key are stored in your browser on this device. Export a backup regularly if the data matters to you.

## Current Default Rules

New vocabulary should include phonetic symbols, Chinese meanings, high-frequency collocations, five examples, word family, and three examples for each related word. English explanations are intentionally removed. Chinese meanings should include both standard dictionary meaning and deeper practical workplace interpretation. Recommendations should avoid obvious derivatives of existing wordbook terms, avoid rare show-off words, prioritize practical Australian workplace use, and reuse saved vocabulary in examples where natural.
